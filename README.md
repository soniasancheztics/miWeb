## 🧠 Descripción

Este proyecto consiste en un sitio web completo que sirve como excelente carta de presentación digital. Cuenta con una navegación intuitiva entre secciones, un apartado visual cuidado y lógica de programación en el cliente para mejorar la experiencia del usuario.

### 🚀 Características Clave
* **Página de Inicio (`index.html`):** Portal de bienvenida con una estructura clara y un diseño visual atractivo.
* **Sección de Contacto (`contacto.html`):** Formulario dinámico con validaciones interactivas antes del envío.
* **Sección "Cómo Llegar" (`comollegar.html`):** Mapa y/o indicaciones optimizadas para que los usuarios te encuentren fácilmente.
* **📱 Diseño Responsive:** Optimizado al 100% para pantallas de smartphones, tablets y ordenadores de escritorio mediante Media Queries y layouts flexibles.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Para la estructura semántica del contenido.
* **CSS3:** Diseños modernos, transiciones suaves, tipografías personalizadas y adaptabilidad móvil.
* **JavaScript (Vanilla JS):** Interactividad, lógica del menú móvil y validación de datos en tiempo real.
